# Image Annotator 预览页面需求文档

## 1. 项目概述

创建一个轻量级的 H5 预览页面，用于实时展示 image-annotator-mcp 工具的所有标注类型和效果。

## 2. 核心目标

- **所见即所得**：每次刷新页面自动渲染所有标注示例
- **无需生成文件**：直接在浏览器中渲染 SVG
- **覆盖所有用法**：展示所有标注类型、颜色、主题和样式变体

## 3. 技术方案

### 实现方式
- 单个 HTML 文件，直接在浏览器中运行
- 使用内联 JavaScript 复刻 `buildSvg` 核心逻辑
- 无需 Node.js 依赖，纯前端实现
- 背景图片使用占位图服务（如 placeholder.com 或 picsum.photos）

### 文件结构
```
image-annotator-mcp/
  └── preview/
      └── index.html    # 预览页面（单文件）
```

## 4. 功能需求

### 4.1 标注类型展示

| 标注类型 | 描述 | 示例数量 |
|---------|------|---------|
| marker/number | 数字标记 | 3种样式（filled/outline/badge） |
| arrow | 箭头 | 实线、虚线、带阴影 |
| curved-arrow | 弯曲箭头 | 多种曲线方向 |
| callout | 标注框 | 4种指针方向（上下左右） |
| rect/rectangle | 矩形框 | 实线、虚线、填充 |
| circle | 圆形 | 实线、虚线、填充 |
| label/text | 文字标签 | 带背景、不带背景、手写体 |
| highlight | 高亮覆盖层 | 不同颜色、透明度 |
| blur | 模糊遮罩 | 不同模糊强度 |
| connector/line | 连接线 | 实线、虚线 |
| icon | 图标徽章 | check/x/warning/info/question |

### 4.2 颜色展示
- 11种预设颜色：red, orange, yellow, green, blue, purple, pink, cyan, teal, white, black
- 语义颜色：success, warning, error, info
- 文档颜色：primary, secondary, accent

### 4.3 主题展示
- documentation（文档风格）
- tutorial（教程风格）
- bugReport（Bug报告风格）
- highlight（高亮风格）

## 5. 页面布局设计

### 5.1 整体结构
```
┌─────────────────────────────────────────┐
│  Header: 标题 + 刷新按钮                 │
├─────────────────────────────────────────┤
│  Section: 颜色表                         │
├─────────────────────────────────────────┤
│  Section: 标注类型示例                   │
│  ┌─────────────────────────────────────┐│
│  │ 示例卡片 1                          ││
│  ├─────────────────────────────────────┤│
│  │ 示例卡片 2                          ││
│  └─────────────────────────────────────┘│
└─────────────────────────────────────────┘
```

### 5.2 设计风格
- 简洁的卡片式布局
- 深色/浅色两种模式（可选）
- 每个示例卡片包含：标题、参数说明、渲染结果
- 背景使用浅灰色网格或渐变，模拟文档风格

## 6. 验收标准

### 功能验收
- [ ] 页面在现代浏览器中正常打开
- [ ] 刷新页面后所有示例重新渲染
- [ ] 覆盖所有 11 种标注类型
- [ ] 展示 11 种预设颜色
- [ ] 展示 4 种主题效果
- [ ] 每种标注类型的变体都有展示

### 视觉验收
- [ ] 示例卡片布局整齐
- [ ] 标注在背景图片上清晰可见
- [ ] 文字说明清晰准确

## 7. 使用方式

直接在浏览器中打开 `preview/index.html` 文件即可。

```bash
# 方式1：直接用浏览器打开
open preview/index.html

# 方式2：使用简单的 HTTP 服务器（如果有）
npx serve preview/
```
